# 📈 MRP API Статистики V2

### Аутентификация API

Некоторые запросы к API можно делать с заголовком **Authentication c MRPAPIKey**, для увеличения лимитов запросов.

> **Не знаете, где взять API ключ?** Инструкция по получению API-ключа доступна на странице [Введение](https://erlc-703b531c.mintlify.app/introduction).

Для просмотра спецификации Stats API V2 используйте OpenAPI файл:

- [stats-v2-erlc-russia-api.yaml](/openapi/stats-v2-erlc-russia-api.yaml)

Подробное описание эндпоинтов будет доступно после размещения OpenAPI spec файла в папке `/openapi/`.

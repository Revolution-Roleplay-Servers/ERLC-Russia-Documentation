### Аутентификация API

Все запросы к API должны содержать валидный заголовок **Authentication**.

**Не знаете, где взять API ключ?** Инструкция по получению API-ключа доступна на странице [Введение](https://erlc-703b531c.mintlify.app/introduction).

### Справка по API

Команда ER:LC Россия и подпроектов (в т.ч. Moscow RolePlay) **не оказывает поддержку** по написанию кода или интеграции API в сторонние решения.

Для просмотра спецификации используйте OpenAPI файл:

- [erlc-russia-server-api.yaml](https://erlc-703b531c.mintlify.app/openapi/erlc-russia-server-api.yaml)

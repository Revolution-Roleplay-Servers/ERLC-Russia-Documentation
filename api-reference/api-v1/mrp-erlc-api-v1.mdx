# 🚗 MRP ER:LC API V1

### Аутентификация API

Все запросы к API должны содержать валидный заголовок **Authentication**.

**Не знаете, где взять API ключ?** Инструкция по получению API-ключа доступна на странице [Введение](/introduction).

### Справка по API

Команда ER:LC Россия и подпроектов (в т.ч. Moscow RolePlay) **не оказывает поддержку** по написанию кода или интеграции API в сторонние решения.

> **POST-эндпоинты имеют другие лимиты запросов. Обязательно читайте и строго соблюдайте все заголовки Rate Limit.**

Для просмотра спецификации API V1 используйте OpenAPI файл:

- [erlc-russia-server-api.yaml](/openapi/erlc-russia-server-api.yaml)

Подробное описание эндпоинтов будет доступно после размещения OpenAPI spec файла в папке `/openapi/`.

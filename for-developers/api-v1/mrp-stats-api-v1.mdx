# 📊 MRP API Статистики V1

### Справка по API

Команда ER:LC Россия и подпроектов (в т.ч. Moscow RolePlay) **не оказывает поддержку** по написанию кода или интеграции API в сторонние решения.

Для просмотра спецификации Stats API V1 используйте OpenAPI файл:
- [stats-erlc-russia-api.yaml](/openapi/stats-erlc-russia-api.yaml)

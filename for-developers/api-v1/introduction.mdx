# 📚 Документация API V1

Этот API предоставляет программный доступ к данным и функциям, связанным с сервером ER:LC и статистикой. Он разработан для интеграции с вашими собственными приложениями и сервисами.

### Структура Документации

1. [MRP ER:LC API V1](/for-developers/api-v1/mrp-erlc-api-v1)
   - Подробное описание всех доступных маршрутов API.
   - Методы, параметры, примеры запросов и ответов.
   - Информация по аутентификации для разных групп эндпоинтов.
2. [MRP API Статистики V1](/for-developers/api-v1/mrp-stats-api-v1)
   - Эндпоинты для получения статистики серверов.
3. [Коды Ошибок](/for-developers/error-codes)
   - Список всех возможных кодов ошибок.
   - Описание причин и способы их решения.
4. [Лимиты Запросов](/for-developers/rate-limits)
   - Список всех возможных Рейт Лимитов.
   - Описание причин и способы их решения.

### Получение API Ключа и Поддержка

Для работы с защищенными эндпоинтами ER:LC API вам потребуется **API ключ**. Чтобы получить свой API ключ, пожалуйста, **откройте тикет** на нашем Discord сервере:

[https://discord.gg/MoscowRus](https://discord.gg/MoscowRus)

Там же вы сможете получить поддержку по любым вопросам, связанным с использованием API.

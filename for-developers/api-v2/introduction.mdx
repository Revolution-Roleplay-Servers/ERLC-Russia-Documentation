# 🚀 Документация API V2

Этот API предоставляет программный доступ к данным и функциям, связанным с сервером ER:LC, статистикой, экономикой и другими сервисами. Он разработан для интеграции с вашими собственными приложениями и сервисами.

### Структура Документации

1. [MRP ER:LC API V2](/for-developers/api-v2/mrp-erlc-api-v2/introduction)
   - Управление сервером: команды, игроки, модерация.
2. [MRP API Экономики V2](/for-developers/api-v2/mrp-civops-api-v2/introduction)
   - Эндпоинты для работы с бизнесами, мафиями, транзакциями.
3. [MRP API Статистики V2](/for-developers/api-v2/mrp-stats-api-v2)
   - Продвинутая статистика серверов.
4. [ClueControl M V2](/for-developers/api-v2/cluecontrol-m-v2)
   - Discord бот для управления сервером.
5. [Коды Ошибок](/for-developers/error-codes)
   - Список всех возможных кодов ошибок.
6. [Лимиты Запросов](/for-developers/rate-limits)
   - Список всех возможных Рейт Лимитов.

### Получение API Ключа и Поддержка

Для работы с защищенными эндпоинтами ER:LC API вам потребуется **API ключ**. Чтобы получить свой API ключ, пожалуйста, **откройте тикет** на нашем Discord сервере:

[https://discord.gg/MoscowRus](https://discord.gg/MoscowRus)

Там же вы сможете получить поддержку по любым вопросам, связанным с использованием API.

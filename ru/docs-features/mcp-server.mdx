## Эндпоинт

```text
https://apidocs.erlcrussia.com/mcp
```

## Что Такое MCP?

Model Context Protocol (MCP) — это открытый стандарт, позволяющий ИИ-ассистентам подключаться к внешним источникам данных и инструментам. Направив MCP-совместимый клиент на наш сервер, ваш ИИ-ассистент сможет искать, извлекать и анализировать информацию из документации API ER:LC Russia в режиме реального времени.

<Tip>
  **MCP-сервер предоставляет доступ к актуальной документации без необходимости вручную копировать и вставлять контент в чат ИИ-ассистента.**
</Tip>

## Подключение

### Claude Desktop

Добавьте следующее в ваш `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "erlcrussia-api-docs": {
      "url": "https://apidocs.erlcrussia.com/mcp"
    }
  }
}
```

### Cursor

Добавьте сервер в настройках MCP вашего Cursor:

```json
{
  "mcpServers": {
    "erlcrussia-api-docs": {
      "url": "https://apidocs.erlcrussia.com/mcp"
    }
  }
}
```

### OpenCode

Добавьте сервер в настройках MCP вашего OpenCode:

```json
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "erlcrussia-api-docs": {
      "type": "remote",
      "url": "https://apidocs.erlcrussia.com/mcp",
      "enabled": true
    }
  }
}
```

### Другие Клиенты

Любой клиент, поддерживающий удалённые MCP-серверы, может быть направлен на указанный выше URL. Точный формат конфигурации см. в документации вашего клиента.

## Доступные Инструменты

После подключения ваш клиент получит доступ к инструментам для:

* **Поиска** по документации по ключевым словам или темам
* **Получения** конкретных страниц и разделов
* **Навигации** по структуре документации

## Зачем Это Использовать?

* Задавайте вопросы своему ИИ-ассистенту об API ER:LC Russia и получайте ответы, основанные на самой актуальной документации
* Избегайте галлюцинаций или устаревшей информации
* Создавайте агентов и рабочие процессы, которые автоматически синхронизируются с изменениями в документации

<Note>
  **MCP-сервер предоставляет доступ только к публичной документации. Для получения ключа API и использования эндпоинтов откройте тикет на нашем Discord-сервере.**
</Note>

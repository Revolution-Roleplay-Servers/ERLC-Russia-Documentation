**ER:LC Russia API** is a powerful developer interface that provides access to our community features and allows real-time interaction with the project's technical system, including game server management, economy, and other capabilities.


## Who can use it?

Any developer working with the ER:LC Russia API can gain access after their ticket is reviewed. An authentication key is issued after the request is approved.
<Note>
  **If you are not a developer and do not work with APIs, this documentation is not for you. Instead, use one of the ready-made partner applications listed below.**
</Note>
